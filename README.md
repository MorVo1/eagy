# eagy - an Apple I brainfuck interpreter
## Download
```bash
$ git clone https://github.com/morvo1/eagy
$ cd eagy
```

## Run
To run it, paste the contents of [eagy.woz](eagy.woz) into any Apple I emulator's rom monitor (preferably into [mine](https://github.com/morvo1/apple1) :wink:).
Then, type in `D014R`, and press enter.
After seeing the `#` prompt, you can start inputing the program. Once you finish, press `;` to start the execution.
